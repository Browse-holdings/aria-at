JAWS

Tester Instructions

  1. Put JAWS into Forms Mode by turning Virtual Cursor off using Insert+Z
  2. Navigate into a checkbox group using the following commands:
    • Tab
    • Shift+Tab

Assertions

PriorityAssertion
must haveRole 'group' is conveyed
must haveGroup name 'Sandwich Condiments' is conveyed
must haveBoundaries of the group (before the first checkbox and after the last checkbox) are conveyed
