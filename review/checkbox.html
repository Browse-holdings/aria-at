Test 1: Navigate through checkbox group in interaction mode.

  • Mode: interaction
  • Applies to: JAWS, VoiceOver, NVDA
  • File: /checkbox/navigate-through-checkbox-group-interaction.html
  • Relevent Specifications:
    • https://w3c.github.io/aria-practices/#checkbox
    • https://w3c.github.io/aria/#checkbox
    • https://w3c.github.io/aria/#aria-checked
    • https://w3c.github.io/aria/#group

JAWS

Instructions

  1. Put JAWS into Forms Mode by turning Virual Cursor off using Insert+Z
  2. Then, perform the task "navigate through boundaries of checkbox group" using the following commands:
    • Tab / Shift+Tab

Assertions

PriorityAssertion
1The role 'group' is conveyed
1The group's name 'Sandwich Condiments' is conveyed
1The boundaries of the group (before the first checkbox and after the last checkbox) are conveyed

VoiceOver

Instructions

  1. Turn Quick Nav off by pressing the Left Arrow and Right Arrow keys at the same time.
  2. Then, perform the task "navigate through boundaries of checkbox group" using the following commands:
    • Tab / Shift+Tab
    • Control+Option+Right / Ctrl+Option+Left
    • Control+Option+A
    • Control+Option+Command+J / Shift+Control+Option+Command+J
    • Control+Option+Command+C / Shift+Control+Option+Command+C
    • Control+U to open the ‘Rotor’ menu; then press the Right arrow until the list of 'Form Controls’ appears; then select a checkbox in the list of form controls using the Up/Down arrow keys; then press ‘Enter’ to navigate to that checkbox

Assertions

PriorityAssertion
1The role 'group' is conveyed
1The group's name 'Sandwich Condiments' is conveyed
1The boundaries of the group (before the first checkbox and after the last checkbox) are conveyed

NVDA

Instructions

  1. Put NVDA into Focus Mode using NVDA+Space
  2. Then, perform the task "navigate through boundaries of checkbox group" using the following commands:
    • Tab / Shift+Tab
